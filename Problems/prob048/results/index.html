Instance Results

These are the results of problem instances given here in this benchmark. Each instance is described using a table specifying the power required to by each agent to communicate with each other agent. Agents that cannot communicate are indicated with a dash (-). The source agent (i.e. the source device in the broadcast tree) is highlighted in red. The optimal result is highlighted in the table as follows: each row of the table represents an agent ai; the table cells marked in yellow indicates that ai is broadcasting to the corresponding other agent in the optimal broadcast tree. The optimal solution cost is the sum of the maximum highlighted value in each row.
