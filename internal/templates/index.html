Overview

CSPLib is a library of test problems for constraint solvers. The library consists of:

  • Problems organised by subject area and problem number

CSPLib also maintains a list of constraint languages and the tools which can be used to solve problems in these languages.

The main motivation for CSPLib is to focus research in constraints away from purely random problems and onto more structured problems.

We currently have {{ num_problems }} problems in {{ num_categories }} categories proposed by {{num_authors}} authors.

Please submit any updates or corrections to CSPLib to our git repository, either as a new issue, or pull request.

The current editors-in-chief of CSPLib are Chris Jefferson and Özgür Akgün. The current website maintainers are Bilal Syed Hussain, Chris Jefferson and Özgür Akgün.

We are grateful to the previous editors of CSPLib: Ian Miguel, Brahim Hnich, Ian Gent and Toby Walsh, and all the contributors to CSPLib since its creation in 1999.
